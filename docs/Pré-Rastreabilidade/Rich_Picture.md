# Rich Picture

## O que é?

Rich Picture é uma técnica visual utilizada na análise de sistemas para representar, de forma livre e informal, a visão geral de um problema, seus atores (quem participa do sistema), operações (o que é feito ou quais ações ocorrem no sistema), armazenamento de dados (onde e como as informações são guardadas), setas (fluxo de informação, comunicação ou influência entre os elementos) e fronteira do sistema (limites do que está sob controle dos desenvolvedores ou do escopo do sistema). Ele é frequentemente usado na fase inicial da modelagem de sistemas.
