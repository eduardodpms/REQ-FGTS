    Entrega 3

    Introdução

    A Entrega 3 tem como objetivo modelar os requisitos elicitados anteriormente. Para isso, foram desenvolvidos os cenários, o diagrama de casos de uso, os léxicos e a especificação suplementar. Com isso, a equipe concluiu a entrega e avançou para a etapa de Inspeção do Grupo +1.

    Apresentação
